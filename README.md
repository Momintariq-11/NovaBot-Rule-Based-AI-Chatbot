# 🤖 NovaBot – GUI-Based Rule-Based Chatbot

NovaBot is a beginner-friendly chatbot built using **Python** and **Tkinter**. It uses simple **rule-based logic** (`if`, `elif`, `else`) to respond to predefined user inputs through an interactive graphical user interface.

## 🚀 Features

- Simple and user-friendly GUI
- Black-themed interface
- Rule-based chatbot responses
- Multiple predefined commands
- Help command
- Exit command
- Beginner-friendly Python code

## 🛠️ Technologies Used

- Python 3
- Tkinter

## 📂 Project Structure

```
NovaBot/
│── novabot.py
```

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Open the project folder.
3. Run the following command:

```bash
python novabot.py
```

For macOS/Linux:

```bash
python3 novabot.py
```
